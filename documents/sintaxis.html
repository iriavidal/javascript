Sentencias

Es una unidad de código que realiza una acción.

Las sentencias son instrucciones que le indican al programa qué hacer. Cada sentencia en JavaScript termina con un punto y coma (;). 

alert("Hola mundo");

Variables

Se utilizan en todos los lenguajes de programación.

Son un espacio en la memoria del ordenador RAM donde se almacenará un valor que podrá cambiar durante la ejecución del programa.

Sintaxis (declaración):

let puntuacion;

Reglas importantes de dar nombre a una variable:

  • Han de comenzar por letra, símbolo $ o _
  • Han de contener solo letras, números, $ y _
  • Son case sensitive (sensible a mayúsculas y minúsculas)
  • No deben ser palabras reservadas
  • Se recomienda que sean descriptivos

Iniciar la variable

  • Es almacenar dentro de ella un valor.
  • let puntuacion; | let num = 0;
  • Se pueden declarar variables en la misma línea. 
    let num, puntuacion, jugador;

¿Qué son?

Es una especie de gran variable en la que podemos almacenar varios datos.

Su ventaja es que puede almacenar un montón de valores.

let articulos = ["zapatillas", "pantalón", "camiseta", "calcetines"];

Se puede guardar en el array cualquier tipo de objetos, aunque no es recomendable.

Acceder a los elementos del array

let arr = ["zapatillas", "camiseta", "pantalón", "calcetines"];
alert(arr[1]); //camiseta

Los índices de los arrays empiezan en 0.

Propiedades y métodos del array importantes

  • length: arr.length //4
  • push: agrega uno o más elementos al final del array y devuelve la nueva longitud del array
  • unshift: agrega uno o más elementos al inicio del array y devuelve la nueva longitud del array
  • pop: elimina el último elemento del array y lo devuelve
  • shift: elimina el primer elemento del array y lo devuelve

Recorrer un array

            const nombres = ["David", "Juan", "Kevin"]; 
            for (let i = 0; i < nombres.length; i++) { 
              console.log(nombres[i]); 
            } 
            // Imprime: 
            // David 
            // Juan 
            // Kevin

            nombres.forEach((elemento) => {
              console.log(elemento);
            });
            //Imprimirá lo mismo que el for

Array.from

Array.from es un método incorporado en JavaScript que se utiliza para crear un nuevo array a partir de un objeto iterable o de una estructura similar a un array. Puede ser una forma más conveniente y expresiva de convertir ciertos tipos de datos en arrays.

Sintaxis: Array.from(iterable[, mapeoFuncion[, contexto]])

  • iterable: el objeto iterable o similar a un array que se convertirá en un nuevo array.
  • mapeoFuncion (opcional): una función de mapeo que se ejecutará para cada elemento del iterable.
  • contexto (opcional): un valor para usar como this al ejecutar la función de mapeo.

Cómo funciona:

  1. Con iterables: Si el objeto pasado a Array.from es iterable (por ejemplo, un array, una cadena, un objeto Set), creará un nuevo array con los elementos de ese iterable.
  2. Con función de mapeo: Puedes proporcionar una función de mapeo como segundo argumento. Esta función se aplicará a cada elemento del iterable antes de ser agregado al nuevo array.

Array.map

El método map se utiliza en arrays y crea un nuevo array con los resultados de llamar a una función proporcionada por cada elemento en el array original.

Es una forma eficiente y expresiva de transformar los elementos de un array sin modificar el array original.

Sintaxis:


    const nuevoArray = arrayOriginal.map(function(elemento, índice, array) {
      // ...lógica de transformación...
      return nuevoElemento;
    });
  • elemento: el elemento actual que está siendo procesado
  • índice: el índice del elemento actual.
  • array: el array original

Diferencia clave entre array.from y array.map

  • Array.from se utiliza para convertir estructuras de datos iterables o similares a arrays en arrays reales.
  • Array.map se utiliza para transformar elementos e un array existente y crea un nuevo array basado en la transformación.

Operador de Propagación (Spread Operator)

Permite expandir expresiones en lugares donde se esperan múltiples argumentos (por ejemplo, funciones) o múltiples elementos (por ejemplo, arrays).

La sintaxis del operador de propagación es ..., seguido de una expresión o iterable (como un array, una cadena, etc.).

En el contexto de funciones:

// Ejemplo con funciones
function sumar(a, b, c) {
  return a + b + c;
}
            
const numeros = [1, 2, 3];
            
// Utilizando el operador de propagación para pasar elementos de un array como argumentos individuales
const resultado = sumar(...numeros);
console.log(resultado); // Salida: 6

En la creación de arrays:

// Ejemplo con arrays
const array1 = [1, 2, 3];
const array2 = [4, 5, 6];
            
// Utilizando el operador de propagación para combinar dos arrays
const arrayConcatenado = [...array1, ...array2];
console.log(arrayConcatenado); // Salida: [1, 2, 3, 4, 5, 6]

.join()

Se utiliza para concatenar los elementos de un array en una sola cadena. La idea principal es crear una cadena donde los elementos del array estén separados por un delimitador específico que determines. 

const frutas = ["manzana", "banana", "cereza"];

// Utilizando join para unir elementos con un guion como delimitador
const cadenaFrutas = frutas.join("-");
            
console.log(cadenaFrutas);
// Salida: "manzana-banana-cereza"

Introducción

Apuntes Javascript
  • Tiene un objetivo concreto: ejecutarse en un navegador
  • Se ejecuta en local, lo cual implica rapidez e inmediatez
  • Es interpretado (no compilado)
  • Es de respuesta inmediata
  • Agrega interactividad a los sitios web
  • Proporciona efectos visuales dinámicos
  • Una librería de código JS ya elaborado, libre, de código abierto disponible para que cualquier programador la utilice en sus proyectos.
  • Simplifica los dos principales inconvenientes de JS:
    • Dificultad a la hora de programar
    • Incompatibilidad entre navegadores
  • JQuery permite agregar funcionalidades avanzadas a nuestros sitios web
  • JQuery se utiliza en millones de sitios web, algunos tan conocidos como los basados en los CMS Drupal, Wordpress, etc
  • ¿Dónde va el código?
    • Dentro de la página. Entre etiquetas <script> y </script>
      • En el head
      • En cualquier otra parte
  • En archivos externos con extensión js
  • Nota: si necesitamos que el script se ejecute después de que se lea todo el html, debemos añadirle el atributo defer

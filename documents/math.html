Números

Apuntes Javascript

Determina cuándo un valor es un número o no.

NaN significa Not a Number

Resultados:

  • true: significa que el valor dado no es un número
  • false: significa que el valor dado sí es un número

parseInt

  • parseInt se utiliza para convertir una cadena de texto en un número entero.
  • Puede tomar dos argumentos: la cadena que se va a convertir y una base numérica opcional que indica en qué sistema numérico se encuentra la cadena (por ejemplo, base 10 para decimal, base 16 para hexadecimal, etc.).
  • Si la base no se especifica, se asume que es 10 en la mayoría de los casos.
  • Se extraen los caracteres numéricos de la cadena hasta que se encuentra un carácter no numérico o se llega al final de la cadena.

parseFloat

  • parseFloat se utiliza para convertir una cadena de texto en un número de punto flotante (número con decimales).
  • Similar a parseInt, pero acepta números decimales y utiliza la notación de punto flotante.
  • Igualmente, puede haber caracteres no numéricos al final de la cadena, y la conversión se detendrá en ese punto.
  • toFixed es un método que se utiliza con números en JavaScript para fijar la cantidad de dígitos decimales después del punto.
  • Se aplica a un número y devuelve una cadena que representa ese número con la cantidad específica de decimales.
  • Toma un argumento que especifica el número de dígitos decimales a mostrar. Este argumento debe estar en el rango de 0 a 20.
  • Redondea el número según las reglas estándar de redondeo.
